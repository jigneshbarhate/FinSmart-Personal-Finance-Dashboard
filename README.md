# 💸 FinSmart – Personal Finance Dashboard

## 🚀 Overview

FinSmart is a modern and responsive **FinTech web application frontend** built using **React.js** and **Tailwind CSS**.
It allows users to visualize their financial data, track transactions, and manage their personal finances through a clean and intuitive UI.


---

## 🎯 Features

* 🏠 **Landing Page**

  * Clean hero section with CTA (Get Started / Login)
  * Feature highlights (Expense Tracking, Budget Goals, Insights)

* 📊 **Dashboard**

  * Total Balance, Income, and Expense cards
  * Recent transactions overview
  * Placeholder for charts (future integration)

* 💰 **Transactions Page**

  * Table view of transactions
  * Search and filter functionality
  * Color-coded entries (Income = Green, Expense = Red)

* 👤 **Profile Page**

  * User info display (UI only)
  * Logout button (UI only)

* 📱 **Fully Responsive Design**

  * Mobile-first layout
  * Collapsible sidebar
  * Responsive navbar

---

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git https://github.com/jigneshbarhate/FinSmart-Personal-Finance-Dashboard
cd finsmart-frontend
```

### 2️⃣ Install Dependencies

```bash
npm install
```

### 3️⃣ Run Development Server

```bash
npm run dev
```

👉 App will run on:

```
http://localhost:5173
```
